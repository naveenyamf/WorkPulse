What's in the Report?
📋 Sheet 1 — Summary
Employee name, email, department, roster assignment, total screenshots, web records, app records, system events
🌐 Sheet 2 — Web Activity
Every URL visited with browser, duration in seconds, and timestamp (up to 10,000 rows)
💻 Sheet 3 — App Usage
App name, total time in HH:MM, number of sessions, first and last seen timestamps
⚡ Sheet 4 — System Events
Startup, shutdown, lock, unlock, sleep, wakeup events with full timestamps and day of week
✉️ SMTP Configuration
📖 Setup Guide
Gmail (recommended)
Host: smtp.gmail.com
Port: 587 (TLS) or 465 (SSL)
User: your.email@gmail.com
Pass: App Password (not your real password)
How to get Gmail App Password:
  1. Go to Google Account → Security
  2. Enable 2-Step Verification
  3. Search "App passwords"
  4. Create one for "Mail"
  5. Use that 16-char code as the password
⚠️ Email is used for Forgot Password only. Enable it so users can reset their passwords without admin help.
