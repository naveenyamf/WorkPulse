✉️ SMTP Configuration
📖 Setup Guide
Gmail (recommended)
Host: smtp.gmail.com
Port: 587 (TLS) or 465 (SSL)
User: your.email@gmail.com
Pass: App Password (not your real password)
How to get Gmail App Password:
  1. Go to Google Account → Security
  2. Enable 2-Step Verification
  3. Search "App passwords"
  4. Create one for "Mail"
  5. Use that 16-char code as the password
⚠️ Email is used for Forgot Password only. Enable it so users can reset their passwords without admin help.
